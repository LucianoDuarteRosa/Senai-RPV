Sistema de Endereçamento

O aplicativo é um sistema de localização que permite aos usuários adicionar produtos, fornecedores, ruas, e prédios, associando a cada produto um endereço. Isso facilita o controle do local físico de cada item, permitindo saber exatamente onde está localizado. O sistema possibilita criar listas de demanda e gera arquivos para impressão em Excel. Ao finalizar uma lista, os produtos são automaticamente baixados. Além disso, há a funcionalidade de importar arquivos CSV com dados de cadastro de produtos e exportar os dados armazenados.
